

# שיפור תצוגת סימים - נגישות, סטטוסים והדפסת הוראות

## מה ישתנה

### 1. תצוגת כרטיסים במקום טבלה (Card View)
במקום טבלה קשה לקריאה, כל סים יוצג ככרטיס צבעוני וברור:
- **ירוק** = תקין וזמין
- **צהוב** = קרוב לפקיעה
- **אדום** = פג תוקף
- **כחול** = מושכר

כל כרטיס יציג: מספר UK, מספר IL, תוקף, חבילה, ושני אינדיקטורים של סטטוס.

### 2. שני אינדיקטורי סטטוס בכל כרטיס
- **סטטוס פורטל** (האתר של CellStation): תקין / קרוב לפקיעה / פג תוקף / מושכר
- **סטטוס מערכת** (המערכת שלך): תואם / צריך החלפה / לא במלאי / מושכר

שניהם יוצגו זה לצד זה עם אייקונים ברורים.

### 3. כפתור הדפסת הוראות חיוג על כל סים
כפתור מדפסת קטן על כל כרטיס סים (שיש לו מספר UK או IL), שמדפיס את הוראות החיוג מיידית - בדיוק כמו במערכת ההשכרות.

### 4. מיון חכם
הסימים יוצגו לפי סדר עדיפות: קודם צריכי החלפה, אחריהם פגי תוקף, קרובי פקיעה, ובסוף תקינים.

## פרטים טכניים

### קבצים שישתנו

| קובץ | שינוי |
|-------|-------|
| `src/pages/CellStation.tsx` | החלפת `SimTable` בקומפוננטת `SimCardGrid` חדשה עם תצוגת כרטיסים, כפתור הדפסה, ושני אינדיקטורי סטטוס |

### מבנה כרטיס סים חדש:
```text
+------------------------------------------+
|  [Badge: חבילה]            [Print btn]   |
|                                          |
|  UK: 447xxxxxxx    IL: 972xxxxxxx        |
|  ICCID: ...123456                        |
|                                          |
|  תוקף: 15/03/2026                       |
|                                          |
|  פורטל: [תקין]    מערכת: [תואם]         |
|  לקוח: שם הלקוח (אם מושכר)              |
+------------------------------------------+
```

### לוגיקת הדפסה:
שימוש ב-`printCallingInstructions` הקיימת מ-`src/lib/callingInstructions.ts`, עם העברת `il_number`, `uk_number` ו-`iccid` מנתוני הסים.

### סדר ביצוע:
1. יצירת קומפוננטת `SimCardGrid` עם כרטיסים צבעוניים
2. הוספת כפתור הדפסת הוראות חיוג
3. הצגת שני סטטוסים (פורטל + מערכת) בכל כרטיס
4. מיון חכם לפי עדיפות
5. שמירת תצוגת הטבלה כאופציה (Toggle) למי שמעדיף
